This directory is for storing image assets for the Gem Collector game.

You can add custom images for:
- Player character
- Different gem types
- Background elements
- UI elements

Image formats supported by Pygame include:
- PNG (recommended for transparency)
- JPG
- BMP
- GIF (non-animated)

Example naming convention:
- player.png
- gem_diamond.png
- gem_ruby.png
- gem_emerald.png
- gem_sapphire.png
- gem_topaz.png
- background.png
- button.png
