This directory is for storing sound assets for the Gem Collector game.

You can add custom sounds for:
- Gem collection
- Level completion
- Game over
- Victory
- Background music

Sound formats supported by Pygame include:
- WAV (recommended for sound effects)
- MP3
- OGG (recommended for music)

Example naming convention:
- collect_diamond.wav
- collect_ruby.wav
- collect_emerald.wav
- collect_sapphire.wav
- collect_topaz.wav
- level_complete.wav
- game_over.wav
- victory.wav
- background_music.ogg
